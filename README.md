# 📝 To-Do List App

Aplicação de lista de tarefas desenvolvida com **React + TypeScript**, com foco em boas práticas, organização de código e experiência do usuário.

## 🚀 Funcionalidades
- Adicionar, remover e concluir tarefas
- Persistência de dados com localStorage
- Tema claro e escuro
- Feedback visual para ações do usuário
- Contador de tarefas concluídas

## 🛠️ Tecnologias utilizadas
- React
- TypeScript
- CSS
